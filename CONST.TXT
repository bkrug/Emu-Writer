       DEF  BLKUSE
       DEF  SPACE,DASH,LF,CR

* Values in this file are constants that
* may need to be shared by different
* modules

* The left-most bit of a memory header
* indicates if the memory chunk is in
* use or not.
BLKUSE DATA >8000

* ASCII codes
SPACE  TEXT ' '
DASH   TEXT '-'
LF     BYTE 10
CR     BYTE 13
       EVEN
       END
